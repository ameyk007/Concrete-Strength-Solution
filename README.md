# 🏗️ Concrete Strength Prediction App

This project is a **Machine Learning Web Application** built using **Streamlit** that predicts the **compressive strength of concrete (MPa)** based on input parameters.

---

## 🚀 Project Overview

Concrete strength depends on multiple factors like cement, water, aggregates, and age.  
This app uses an **XGBoost Regression Model** to predict strength accurately.

---

## 🧠 Model Details

- Algorithm: **XGBoost Regressor**
- Problem Type: Regression
- Evaluation Metrics:
  - Mean Squared Error (MSE)
  - R² Score

---

## 📊 Input Features

The model takes the following inputs:

- Cement
- Blast Furnace Slag
- Fly Ash
- Water
- Superplasticizer
- Coarse Aggregate
- Fine Aggregate
- Age (in days)

---

## 🎯 Output

- Predicted **Concrete Compressive Strength (MPa)**
